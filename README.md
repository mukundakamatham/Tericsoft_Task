## Run Locally

Clone the project

```bash
  git clone https://github.com/mukundakamatham/Tericsoft_Task.git
```

Go to the project directory

```bash
  cd Tericsoft_Task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
